Response time

Asrnaam is a small project. We answer most messages within a week. Please include the name slug or page URL when reporting a correction so we can find it quickly.

Useful subject lines

  • Correction — for factual errors in a meaning, root, Quranic citation, or Urdu form.
  • Famous bearer — to suggest a historical or contemporary figure for a name's Famous Bearers section.
  • New name — to propose a canonical name we don't yet cover.
  • Source — to ask which classical authority a particular letter reading rests on.
